# SRegionProtector
SRegionProtector is a GoMint plugin that allows players to protect their regions.
## Download
* [GoogleDrive](https://drive.google.com/file/d/13Nve2R6Cp3GbwLe-VvsYHVxDD0JeNS11/view?usp=sharing)
## Features
* API for another plugins
* Lots of flags
* Flexible settings
* High performance
