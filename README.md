# SRegionProtector
SRegionProtector is a GoMint plugin that allows players to protect their regions.
## Download
* [GoogleDrive](https://drive.google.com/file/d/1zc-zYi1iIMVQAN8_u6KMuVpi3-lpsdh7/view?usp=sharing)
## Features
* API for another plugins
* Lots of flags
* Flexible settings
* High performance
